# Java Learning Repository

这是一个系统化的 Java 学习仓库，适合零基础或想夯实基础的同学。

## 🎯 学习目标

- 掌握 Java 核心语法和面向对象编程
- 熟练使用 Java 集合框架、IO、多线程等
- 理解 JVM 原理和常用设计模式
- 为后续学习 Spring Boot、微服务等打好基础

## 📁 仓库目录结构

```bash
java-learning/
├── README.md                 # 本文件
├── basics/                   # Java 基础语法
├── oop/                      # 面向对象编程
├── collections/              # 集合框架
├── exceptions/               # 异常处理
├── io/                       # 文件 I/O
├── multithreading/           # 多线程
├── jdbc/                     # 数据库操作
├── jvm/                      # JVM 原理
├── design-patterns/          # 设计模式
├── projects/                 # 实战小项目
└── resources/                # 学习资源
```

## 🚀 快速开始

1. **克隆仓库**
   ```bash
   git clone https://github.com/zhou420476-lab/java-learning.git
   ```

2. **进入目录**
   ```bash
   cd java-learning
   ```

3. **使用 IntelliJ IDEA 或 Eclipse 打开**

## 📖 学习路线（推荐顺序）

1. **Java 基础** → 2. **面向对象** → 3. **集合框架** → 4. **异常与 IO** → 5. **多线程** → 6. **JVM** → 7. **实战项目**

## 🛠️ 开发环境推荐

- **JDK**: 17 或 21（推荐 LTS 版本）
- **IDE**: IntelliJ IDEA Community（免费）
- **构建工具**: Maven 或 Gradle

## 📚 学习资源

- 官方文档: [Java SE 文档](https://docs.oracle.com/en/java/)
- Bilibili 推荐教程（搜索关键词）
- 《Java核心技术卷 I》
- 《Effective Java》
